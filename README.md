An interactive web application for creating educational physical maps with customizable geographical features. Perfect for high school geography students and teachers.

## Features

- **Multiple Map Templates**: Pre-configured regions (South America, Europe, Africa, Asia, North America)
- **Geographical Features**: Add mountains, rivers, lakes, forests, deserts, cities, and coastlines
- **Customizable Styling**: Choose colors for each feature
- **Real-time Preview**: See your map as you build it
- **Legend Generation**: Automatic legend creation
- **Easy Download**: Export maps as PNG images
